Vefsíðugerð

Fyrsta verkefni námskeiðsins er að setja upp vefsíðu sem hýsir framtíðar verkefni og ferilskrá

Undirbúningur

Ég byrjaði á að horfa á kynningar myndbandið um verkefnið og þar sem ég hafði smá reynslu á html skriftum var ég ekki lengi að hefjast handa. Ég byrjaði á því að hlaða niður Brackets og Gitbash þar sem mér fannst "Live" eiginleikinn og "Git intergration" hjá því aðdragandi. Svo fór ég að skoða sniðmát fyrir vefsíðuna og fann eitthvað sem mér leist vel á og var það "Forty" eftir HTML5UP því það var einfalt og skilvirkt uppsett og er það sniðmát þessarar vefsíðu.

Uppsetning

Það fyrsta sem ég gerði var að fara yfir sniðmátið og aðlaga það að mínum þörfum, ég tók út það sem mér fannst ekki passa og bætti því sem vantaði. Mér fannst mikilvægt að hafa aðgengilegt valmyndarkerfi (menu) sem gerir notendum auðvelt að fara á milli síðna á vefsíðunni. Þetta hjálpar til við að halda síðunni skipulagðri og bætir heildarupplifunina með skýru og einföldu flæði. Ég breytti ekki litarþemanu því mér fannst það nokkuð flott en setti inn nýja mynd undir verkefni 1, það var skjáskot af html kóðanum og kom það mjög vel út með myndina alveg óunna. Helsta viðbótin sem ég setti inn er ferilskráin.

Ferilskrá

Þegar kom að því að setja inn ferilskrána mína vildi ég að hún væri aðgengileg sem PDF-skrá, svo að notendur gætu auðveldlega skoðað hana án þess að þurfa að hlaða henni niður. Til að framkvæma þetta þurfti ég að finna lausn fyrir innbyggingu á PDF-skjölum í HTML. Eftir að hafa leitað á Google fann ég út að best væri að nota tög til að birta PDF-skjalið beint á síðunni. Ég byrjaði á því að bæta PDF-skránni við verkefnamöppuna og skrifaði síðan eftirfarandi kóða til að birta hana á vefsíðunni: 

 <embed> style="width:90%;height:100%" src="CV_EGA.pdf" type="application/pdf"> </embed>
Þar næst fínstillti ég stærðina á skjánum með því að stilla width og height, þar sem markmiðið var að ferilskráin yrði þægileg í skoðun án þess að hún þyrfti að fara úr skjánum.

Github og git bash uppsetning

Uppsetning á github og git bash var gert skref fyrir skref eftir kennslumyndbandi. Fyrst og fremst gerði ég reposatory undir nafninu TSF_EGA, þar er þessi vefsíða hýst og til þess að setja inn allar skrár og auðvelda framtíðar uppfærslur á þessari vefsíðu notaðist ég við git bash og notaði eftirfarandi skipanir:

git init
git remote add origin 
git add .
git commit -m "Initial commit"
git push -u origin main

Svo með þessum skipunum var búið að setja allar skrárnar inn í repository-ið á github og þá fann ég linkinn fyrir vefsíðuna í "pages" glugganum.

Áskoranir og lausnir

Ein helsta áskorunin var að stilla PDF-skjalið þannig að það væri aðgengilegt á öllum skjástærðum, sérstaklega fyrir farsíma. Til að leysa þetta þurfti ég að prófa ýmsar stærðastillingar og tryggja að síða væri samhæfð við mismunandi tæki. Auk þess tók smávægilegur tími að átta mig á hvernig ætti að hlaða síðunni upp á GitHub með réttri uppsetningu.
